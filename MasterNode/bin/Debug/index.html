Как это работает?

Master узел принимает HTTP запросы и распределяет задачи
Slave узлы обрабатывают текст с помощью алгоритма TextRank
Протокол UDP обеспечивает быструю связь между узлами
TextRank извлекает важные предложения с помощью графового анализа
